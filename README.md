# My Game Works（Web App）

## 概要
My Game Works は、プレイしたゲーム情報を JSON データとして管理し、  
一覧・ソート・レビュー表示を行うシンプルな Web アプリです。

---

## MVP（Minimum Viable Product）
本アプリの MVP は以下の機能を満たすことです。

### 1. ゲームレビュー一覧表示
- ゲーム名
- 評価
- ジャンル
- 価格
- ひとことレビュー
- スクリーンショット

をカード形式で一覧表示できること。

---

### 2. ソート機能
以下のうち、最低 1 種類以上の並び替えが可能。
- 評価順
- 新着順
- プレイ時間順

---

### 3. JSON によるデータ管理
ゲーム情報を JSON 形式で管理し、
- ゲームの追加
- 情報の編集

が簡単に行える構成になっていること。

---

### 4. レスポンシブ対応
PC・スマートフォンのどちらでも閲覧可能。

---

## 主な機能
- ゲームカード一覧表示
- 評価・ジャンル・価格の表示
- ソート機能
- スクリーンショットの拡大表示

---

## 使用技術
- HTML / CSS / JavaScript
- ChatGPT5.1
- GitHub Pages

---

## 制作目的
- Web アプリの基本構造理解
- PC買ったばかりの友達におすすめゲームを伝えたい
